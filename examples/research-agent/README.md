# research-agent

A small research brief agent.

It demonstrates:

- `instructions.md` for synthesis structure
- `skills/source-quality.md` for evidence discipline
- `tools/search_sources.ts` for typed source lookup

Run from the repo root:

```bash
ANTHROPIC_API_KEY=sk-ant-... pnpm demo:research
```

Try:

- `Research why folder-based agents are easier to maintain.`
- `What evidence supports evals as a pre-deploy habit?`
