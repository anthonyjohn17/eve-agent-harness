# Reporting Security Issues

If you believe you have found a security vulnerability in **Eve Agent Harness**
or the bundled eve framework tree, please report it privately.

Please use the repository's private security advisory flow when available.

Do not open public issues for security vulnerabilities.

We will investigate legitimate reports and do our best to fix confirmed problems
quickly.

Upstream eve / Vercel-related vulnerabilities in the published npm `eve` package
may also be reportable through upstream channels when applicable.
