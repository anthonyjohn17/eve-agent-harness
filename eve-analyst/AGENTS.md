# eve Agent App

This project uses the eve framework inside **[Eve Agent Harness](../README.md)**.

Before writing code, read the relevant guide from the installed eve package
docs. In most installs, those docs are at `node_modules/eve/docs/`. In
workspaces or local package installs, resolve the installed `eve` package
location first and read its `docs/` directory. If package docs are unavailable,
use https://eve.dev/docs as a fallback.

Also see:

- [`../AGENTS.md`](../AGENTS.md) — distribution-level coding-agent guidance
- [`../docs/AGENT_CONVENTION.md`](../docs/AGENT_CONVENTION.md) — folder slots
- [`SPEC.md`](./SPEC.md) — this agent's design contract
